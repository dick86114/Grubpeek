# GrubPeek 使用手册

欢迎使用 **GrubPeek（今天有什么好吃的）**！本手册将指导您如何查看菜单以及如何管理后台数据。

## 1. 访问系统
请通过浏览器访问系统地址（例如：`http://localhost:3000` 或公司内网地址）。

## 2. 普通用户指南

### 2.1 查看今日菜单
进入首页后，您将默认看到**今天**的菜单信息。
页面主体包含四个卡片区域：
-   **早餐** (07:30 - 09:00)
-   **午餐** (11:30 - 13:00)
-   **晚餐** (17:30 - 19:00)
-   **外卖包点** (14:00 - 19:00)

每个卡片中列出了具体的菜品分类和名称。
*   带有 ⭐ 星号标记或高亮显示的菜品为**特色推荐菜**。
*   卡片右上角显示该餐别的固定价格（如有）。

### 2.2 查看其他日期
页面左侧（或顶部）有一个**日历控件**。
-   点击日历上的日期，右侧菜单区域将即时刷新显示该日期的菜单。
-   日历上带有颜色标记的日期表示系统中有该天的菜单数据。
-   点击日历顶部的“<”或“>”箭头可切换月份。

### 2.3 切换主题模式
点击页面右上角的 **设置图标 (⚙️)** 或下拉菜单：
-   选择 **日间**：强制使用浅色温馨主题。
-   选择 **夜间**：强制使用深色护眼主题。
-   选择 **跟随系统**：自动匹配您操作系统的颜色设置。

## 3. 管理员指南

### 3.1 进入管理后台
1.  在首页右上角点击设置图标，选择 **“进入后台管理”**。
2.  输入管理员密码（初始密码请咨询IT部门或查看部署配置），点击确认。

### 3.2 导入菜单
这是最常用的功能。您只需要上传 Excel 文件即可。
1.  在管理后台左侧菜单选择 **“文件管理”**。
2.  点击右上角的 **“上传菜单文件”** 按钮。
3.  选择本地的 `.xlsx` 或 `.et` 格式菜单文件。
    *   *注意：文件名建议包含日期范围，如 `省投食堂菜单：2026年1月4日-9日.xlsx`，以便系统自动识别。*
4.  上传成功后，系统会自动解析并覆盖对应日期的旧数据。

### 3.3 管理已上传文件
在“文件管理”列表中，您可以：
-   查看已上传的文件名和上传时间。
-   点击 **“删除”** 按钮：这将删除该文件以及**该文件所包含的所有菜单数据**。请谨慎操作。
-   点击 **“下载”** 按钮：下载原始 Excel 文件。

### 3.4 编辑每日菜单
如果发现导入的数据有误，可以进行微调：
1.  在管理后台左侧菜单选择 **“菜单管理”**。
2.  在顶部日期选择器中选择要编辑的日期。
3.  下方将列出当天的所有菜品。
4.  您可以 **修改** 菜名、价格、分类，或者 **删除** 某条记录。
5.  也可以点击 **“新增菜品”** 手动添加一条记录。

### 3.5 修改密码
1.  在管理后台右上角点击 **“设置”**（锁图标）。
2.  输入新密码并确认。
3.  下次登录请使用新密码。

## 4. 常见问题 (FAQ)

**Q: 为什么日历上某天是灰色的/点击后显示“暂无供应”？**
A: 说明管理员尚未上传包含该日期的菜单文件，或者当天确实不提供餐食（如节假日）。

**Q: 手机上可以看吗？**
A: 可以。系统已做适配，手机浏览器访问体验良好。

**Q: 上传文件失败怎么办？**
A: 请检查：
1. 文件格式是否为 `.xlsx` 或 `.et`。
2. 文件内容格式是否符合模板规范（包含“早餐”、“午餐”等关键字）。
3. 文件名中是否包含可识别的年份和日期信息。
