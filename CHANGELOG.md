# 更新日志 (Changelog)

本项目的所有显著更改都将记录在此文件中。

## [v1.0.0] - 2025-01-18

### 🎉 新增功能 (Added)
*   **后台管理系统**：
    *   新增 `/admin` 管理页面，支持密码验证。
    *   实现了 Excel 菜单文件的上传、下载和删除功能。
    *   新增菜单数据在线查看与编辑功能。
    *   实现了文件删除时自动清理关联数据库记录的逻辑。
*   **用户界面优化**：
    *   首页菜单卡片布局优化，解决在不同屏幕尺寸下的堆叠问题。
    *   优化了日期选择器的样式，使其更加美观易用。
    *   调整了菜品名称列宽，提升阅读体验。
    *   集成了管理入口和退出登录功能到顶部导航栏。
*   **系统配置**：
    *   新增基于 SVG 的动态 Favicon (`Utensils` 图标)。
    *   更新浏览器标签页标题为 "今天有什么好吃的"。
    *   隐藏了 Next.js 左下角的开发环境指示器 (Dev Indicator)。

### 🛠 架构调整 (Changed)
*   **目录重构**：
    *   将脚本文件移动至 `scripts/` 目录。
    *   将类型定义文件统一移动至 `web/types/index.ts`。
    *   创建 `docs/` 目录并完善了全套项目文档。
*   **技术栈升级**：
    *   全面适配 Next.js 16 和 React 19。
    *   使用 Tailwind CSS v4 进行样式管理。

### 🐛 问题修复 (Fixed)
*   修复了首页卡片在窄屏下显示异常的问题。
*   修复了日期选择器样式错乱的问题。
*   修复了 Next.js 开发指示器无法通过配置关闭的问题（通过 CSS 强制隐藏）。
